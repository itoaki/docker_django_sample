# dockerでdjangoの開発環境を作成するためのリポジトリです

## 1.git clone

~~~
git clone git@github.com:itoaki/docker_django_sample.git
~~~

## 2.exec sh

~~~
cd docker_django_sample
sh setting.sh
~~~

## 3.docker-compose up!

~~~
sh start.sh
~~~

## 4.Access localhost on browser
http://localhost:8000/

## if you wont to stop container

~~~
sh stop.sh
~~~
